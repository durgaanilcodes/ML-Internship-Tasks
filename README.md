# ML Internship Tasks — Codveda Technology

##  Project Overview

This project demonstrates the implementation of fundamental Machine Learning algorithms using Python and Scikit-Learn. It includes data preprocessing, regression, classification, and ensemble learning techniques applied on a housing dataset.

The goal is to build, train, and evaluate ML models while understanding real-world workflow.

##  Tasks Completed


##  Level 1 — Basic Machine Learning

###  Task 1: Data Preprocessing

* Handled missing values
* Encoded categorical variables
* Normalized data
* Split dataset into train/test sets



###  Task 2: Linear Regression

* Model: Linear Regression
* Dataset: Housing dataset
* Goal: Predict house prices
* Evaluation:

  * Mean Squared Error (MSE)
  * R² Score


###  Task 3: K-Nearest Neighbors (KNN)

* Classification model
* Evaluated using:

  * Accuracy
  * Confusion Matrix
  * Classification Report


##  Level 2 — Intermediate Machine Learning

###  Task 1: Logistic Regression

* Converted regression dataset into classification
* Predicted high/low house prices
* Evaluated using:

  * Accuracy Score
  * Precision, Recall, F1-score



###  Task 2: Decision Tree Classifier

* Built decision tree model
* Visualized tree structure
* Controlled overfitting using max_depth
* Evaluated performance metrics



##  Level 3 — Advanced Machine Learning

###  Task 1: Random Forest Classifier

* Ensemble learning model
* Improved accuracy using multiple decision trees
* Feature importance analysis performed
* Evaluated using classification metrics
